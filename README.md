# MicroPulse

MicroPulse is an AI-powered hyperlocal demand intelligence platform built for Bharatâ€™s retail ecosystem.

It predicts SKU-level demand at PIN-code level using historical sales, weather signals, and festival intelligence.

## Problem
Retailers struggle with stock-outs and overstocking due to lack of hyperlocal demand visibility.

## Solution
MicroPulse integrates multiple real-world signals and converts AI forecasts into actionable stocking recommendations.

Developed for AWS AI for Bharat Hackathon.

## Architecture Overview

MicroPulse follows a modular architecture:

1. Data Layer
2. AI Forecasting Layer
3. Recommendation Engine
4. Dashboard Interface

Built using AWS (S3, SageMaker, Lambda) and Python-based ML models.
